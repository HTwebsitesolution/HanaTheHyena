# Hana Activity Pack - Cover Page Design Specification

## File Information
- **Filename**: `hana-activity-pack-all-in-one.pdf`
- **First Page**: Cover (this specification)
- **Following Pages**: 6 individual activities in order (see below)

## Page Specifications
- **Size**: A4 portrait (210 × 297 mm)
- **Margins**: 20 mm all around (generous for "book-like" feel)
- **Background**: White or very light cream (#FEF9E7 or #FFFBF0)
- **Print-friendly**: Designed for black and white printing (color accents optional)

## Layout Structure

### Top Area (Discussion Guide Blurb + Logo + Title)

**Discussion Guide Blurb** (At the very top):
- **Position**: Top center, approximately 15-20mm from top
- **Text**: "Includes a one-page Discussion Guide for parents and teachers to help you talk about courage, belonging, and kindness after reading Hana's story."
- **Font**: Sans-serif, italic or regular
- **Size**: 9-10pt
- **Color**: #78716C (medium stone gray) or #059669 (emerald-600)
- **Style**: Subtle, not too prominent
- **Max width**: 160mm (centered)
- **Position**: Above logo/title area

**Logo/Image**:
- Small Hana logo OR cropped circle portrait of Hana
- Size: Approximately 40-50mm diameter
- Position: Centered horizontally
- **Position**: Approximately 10-15mm below discussion guide blurb

**Title** (Below logo, centered):
- **Line 1**: "HANA THE HYENA"
  - Font: Friendly display font (similar to book title)
  - Size: 28-32pt
  - Weight: Bold
  - Color: #1C1917 (dark stone/black)
  - Letter spacing: Slightly increased for readability

- **Line 2**: "Activity Pack"
  - Font: Clean sans-serif (Arial, Helvetica, or similar)
  - Size: 18-20pt
  - Weight: Regular or Medium
  - Color: #78716C (medium stone gray)
  - Position: 8-10mm below "HANA THE HYENA"

### Middle Area (Tagline + Description)

**Tagline** (Centered, below title):
- Text: "Fun activities based on *The Hyena Who Went To Dinner*"
- Font: Sans-serif, italic for book title
- Size: 14-16pt
- Color: #57534E (stone gray)
- Position: 20-25mm below "Activity Pack"

**Description** (Centered, 2-3 lines):
- Text:
  ```
  Colouring pages, puzzles, and creative worksheets to explore 
  Hana's Ethiopian adventure with children aged 6 and up.
  Perfect for homes, classrooms, and libraries.
  ```
- Font: Sans-serif, clean and readable
- Size: 11-12pt
- Color: #57534E (stone gray)
- Line height: 1.5
- Max width: 160mm (centered)
- Position: 15-20mm below tagline

### Activity Overview (Lower-Middle)

**Section Title**:
- Text: "What's inside"
- Font: Sans-serif, bold
- Size: 14-16pt
- Color: #1C1917 (dark stone)
- Position: 20-25mm below description

**Activity List** (Two columns or rounded box):
- Layout: Two columns, centered
- Background: Optional light box (#FEF3C7 - amber-50) with rounded corners
- Border: Optional thin border (#FCD34D - amber-300)
- Padding: 8-10mm inside box
- Font: Sans-serif
- Size: 10-11pt
- Color: #1C1917 (dark stone)
- Line height: 1.6
- Column spacing: 15-20mm between columns

**List Items** (6 activities):
1. Hana's Big Dreams colouring page
2. Dinner with New Friends colouring page
3. Help Hana Find the City (maze)
4. Spot the Differences – On the Way to Addis
5. Hana's Feelings Journey timeline
6. Postcard from Hana writing prompt

**Bullet Style**: 
- Small circle or dot (●) or em dash (—)
- Color: #059669 (emerald-600) or #D97706 (amber-600)

## Visual Elements

### Border/Frame (Optional)
- **Style**: Thin rounded rectangle
- **Color**: #FCD34D (amber-300) or #10B981 (emerald-500)
- **Thickness**: 1-2pt
- **Corner radius**: 8-10mm
- **Position**: 5mm inset from page edges (creates 15mm margin inside border)

### Decorative Elements

**Option 1 - Border Strip**:
- Thin horizontal strip at top (10-15mm height)
- Color: #FCD34D (amber-300) or gradient amber to emerald
- Pattern: Optional subtle tile-inspired pattern

**Option 2 - Bottom Illustration**:
- Small illustration of Hana walking along bottom edge
- Size: Approximately 60-80mm wide
- Position: Bottom left or bottom center
- Style: Line art, light gray (#D6D3D1) so it doesn't drain ink
- Opacity: 30-40% for subtle effect

**Option 3 - Stack of Books**:
- Small illustration of books with Hana sitting next to them
- Position: Bottom right corner
- Size: Approximately 50-60mm
- Style: Line art, light gray

### Background Pattern (Optional)
- Very subtle pattern if desired
- Opacity: 5-10% maximum
- Pattern: Ethiopian-inspired geometric shapes OR simple dots
- Color: #FCD34D (amber-300) or #D6D3D1 (stone-300)

## Color Palette (Print-Safe)
- **Primary Dark**: #1C1917 (stone-900) - Text
- **Secondary**: #57534E (stone-600) - Secondary text
- **Accent**: #D97706 (amber-600) - Highlights, borders
- **Accent 2**: #059669 (emerald-600) - Bullets, accents
- **Background**: #FFFFFF or #FEF9E7 (white or cream)
- **Light Background**: #FEF3C7 (amber-50) - Box backgrounds

## Typography Guidelines

### Display Font (Title)
- **Options**: 
  - Playfair Display (serif, elegant)
  - Quicksand (rounded, friendly)
  - Comfortaa (soft, approachable)
  - Or match the book's title font if available

### Body Font (Sans-serif)
- **Options**:
  - Arial
  - Helvetica
  - Inter
  - Open Sans
  - System fonts for compatibility

## Page Order in Complete Pack

After the cover page, include activities in this order:

1. **Cover Page** (this specification)
2. Hana's Big Dreams colouring page
3. Dinner with New Friends colouring page
4. Journey maze (Help Hana Find the City)
5. Spot the difference (On the Way to Addis)
6. Feelings timeline (Hana's Feelings Journey)
7. Postcard from Hana writing prompt

## Optional: Teacher/Parent Overview Page

If including `hana-activity-overview.pdf` content:
- **Position**: After cover, before activities OR as last page
- **Content**: 
  - "What's inside" list (same as cover)
  - Theme connections for each activity
  - Suggested age ranges
  - Quick tips for use

## Design Principles

1. **Print-Friendly**: All elements should work in black and white
2. **Professional**: Clean, organized, intentional design
3. **Kid-Appealing**: Friendly fonts, approachable layout
4. **Educational**: Clear structure, easy to navigate
5. **Branded**: Consistent with Habesha Press and Hana branding
6. **Ink-Efficient**: Minimal use of heavy fills or dark backgrounds

## Implementation Notes

- Use design software: Canva, Adobe InDesign, or Illustrator
- Export as high-resolution PDF (300 DPI minimum)
- Test print on standard A4 paper
- Ensure all text is readable when printed in grayscale
- Verify margins are sufficient for binding/stabling if needed

## Checklist Before Finalizing

- [ ] Logo/illustration placed and sized correctly
- [ ] All text is readable and properly sized
- [ ] Margins are consistent (20mm all around)
- [ ] Colors work in grayscale/black and white
- [ ] Activity list matches actual contents
- [ ] Credits and usage notice are accurate
- [ ] Overall layout is balanced and professional
- [ ] Test print looks good on standard printer
